Features

FAQ

Does CatsuBrowser work without WebView2?

No. On Windows 7/8/10/11, install WebView2 manually.

Is the project 100% C#?

Yes, the browser is fully developed in C# (WPF).

Can I contribute?

Yes, open issues or pull requests on GitHub.
