Navigateur léger, rapide et compatible Windows 7

CatsuBrowser est un navigateur WPF en 32 bits basé sur .NET Framework 4.7.2 et WebView2. Il démarre vite, consomme très peu de mémoire et reste simple à utiliser.

Fonctionnalités

Multi‑onglets

Ouvrez, fermez et organisez vos onglets facilement, chaque onglet ayant sa navigation indépendante.

Contrôles de navigation

Boutons Précédent, Suivant, Actualiser et Accueil pour une navigation fluide.

Barre d’adresse intelligente

Saisissez une URL ou un terme de recherche, et ouvrez automatiquement la recherche Google.

Historique par onglet

Consultez, réouvrez ou supprimez des entrées, et nettoyez l’historique en un clic.

Interface moderne

Thème sombre inspiré d’Aero/Fluent Design, avec composants épurés et réactifs.

Télécharger

Installez la dernière version stable de CatsuBrowser via les Releases GitHub.

Dernière release Toutes les versions
  • Format: Installeur EXE ou MSI pour Windows.
  • Architecture: 32 bits (x86).
  • Dépendance: Microsoft Edge WebView2 Runtime.

Compatibilité et prérequis

  • Windows: Windows 7 ou version ultérieure.
  • .NET: .NET Framework 4.7.2.
  • Runtime: Edge WebView2 Runtime installé.
  • IDE: Visual Studio 2019 ou version ultérieure pour compiler.

Roadmap

  • Favoris: Gestion des bookmarks.
  • Paramètres: Page des préférences (page d’accueil, thème).
  • Packaging: MSIX pour l’intégration navigateur par défaut.
  • Extensions: Support basique des extensions.
